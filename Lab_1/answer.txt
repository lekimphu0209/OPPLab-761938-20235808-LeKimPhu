Questions: 
What happens if users choose “Cancel”? 
=> Nếu bấm Cancel -> option = JOptionPane.CANCEL_OPTION = 2
=> câu lệnh (option==JOptionPane.YES_OPTION?"Yes":"No") sẽ có option khác với YES_OPTION nên in ra No
 
How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” 
and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help). 
=> Do câu lệnh sư dụng là JOptionPane.showConfirmDialog(null , "do you want to change to the first class ticket?"); nên Java sẽ chọn optionType mặc định là YES_NO_CANCEL_OPTIOB để truyền vào
=> Nếu muốn hiển thị 2 chooses là YES và NO , chỉ cần truyền vào optionType là YES_NO_OPTION
=> câu lệnh mới : 
JOptionPane.showConfirmDialog(
    null,
    "Message",
    "Title",
    JOptionPane.YES_NO_CANCEL_OPTION
);
